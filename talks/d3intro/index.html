What is D3?

D3 is a Javascript library for manipulating documents based on data.

  • Most robust library.
  • Steep learning curve - tutorials + exercises recommended!
    • DashingD3Tutorials.
    • Let's Make A Bar Chart.
  • Simpler alternatives: Chart.js, plot.ly.

Code Sample For A Simple Bar Chart


// Sets the scale for the x-axis
var x = d3.scale.linear().domain([0, d3.max(data)]).range([0, width]);					
// Sets the dimensions of the chart container
var chart = d3.select(".chart")
    .attr("width", width)
    .attr("height", barHeight * data.length);
// For each data entry, attaches a graphic object to the chart
var bar = chart.selectAll("g")
    .data(data)
    .enter().append("g")
    .attr("transform", 
    	function(d, i) { return "translate(0," + i * barHeight + ")"; });
// For each graphic object, appends a rectangle with width and height attributes
bar.append("rect")
    .attr("width", x)
    .attr("height", barHeight - 1);

Where is D3 used?

  • Researchers display data in their papers and journals.
  • Corporate institutions display interactive data on their websites.
  • Data scientists communicate their data to clients/stakeholders.

As web devs, you may interact with one or more of such professionals depending on the organisation you work for.
